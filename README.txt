
# Commercial Lending GenAI Agent ETL Hackathon Bundle

This bundle contains **synthetic** commercial lending data and schemas to build, test, and validate GenAI ETL agents that transform a source operational schema into a BigQuery analytical star schema.


## Notes
- All data is synthetic and randomly generated for hackathon purposes.
- Dates span 2022-01-01 to 2025-09-30.
- Floating rates include SOFR, Prime, EURIBOR, SONIA, TIIE, BBSW with tenor variants.
